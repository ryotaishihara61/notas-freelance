# 修正指示：不要なpreload設定の削除

ブラウザのコンソールに「preloaded but not used」という警告が出ています。
これは、手動または自動で設定された `<link rel="preload">` が、実際の画像表示とマッチしていないためです。

## 対応内容
1. `src/app/layout.tsx` または `src/app/page.tsx` 内に、手動で `<link rel="preload" ...>` というタグが書かれていないか確認してください。もしあれば、それを削除してください。

2. もし手動での記述がない場合、`next.config.ts` の設定などによる自動生成の可能性が高いですが、現状実害はないため、特にコードの変更は行わず、ファイル内のコメントアウトなどで対応可能であれば調整してください。

基本的には「手動で書かれた link preload タグ」があれば削除するだけでOKです。
